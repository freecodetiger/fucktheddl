# DDL Agent v0.2.1 正式版

v0.2.1 是一个本地优先的正式版本。应用不再依赖项目后端登录服务，日程和待办保存在手机本地，AI 与语音能力由用户在设置中填写自己的 DeepSeek 与阿里云配置来驱动。

## 主要更新

- 去掉登录流程和后端 API 主链路，应用启动后直接进入本地工作区。
- 新增本地 Agent 客户端，支持创建、修改、删除、查询日程和待办。
- 语音识别文本会在停顿后继续累积，不再覆盖前面已经说过的内容。
- 优化语音输入后的备注理解，并区分新一轮对话和上一轮对话上下文。
- 首页、日历、待办加入物理滑轨切换，底部菜单点击也会平滑切换。
- 为主要按钮、时间选择等高频操作加入更清脆的触感反馈。
- 统计页改为两个翻页小日历，展示已完成日程数和已完成待办数。
- 新增每日一次提醒，可在设置中开启并选择时间；通知只显示今天未过期事项，不展示逾期项目。

## 安装与覆盖升级

- 安装包：`DDLAgent-v0.2.1.apk`
- Android 会按同一个 `applicationId` 和同一套 release 签名识别升级。
- 从 v0.2.0 升级到 v0.2.1 时，`versionCode` 已从 `7` 提升到 `8`，可直接覆盖安装。

## 注意事项

- Android 13 及以上需要授予通知权限后，每日提醒才能正常弹出。
- 每日提醒使用普通 Android 后台调度，不申请精确闹钟权限；部分系统可能会因省电策略延迟提醒。
- 如果用户手动强行停止应用，系统可能不会继续投递后台提醒，需要重新打开应用。
