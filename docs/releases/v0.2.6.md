# DDL Agent v0.2.6 正式版

v0.2.6 提升了每日提醒的可靠性。应用会优先使用闹钟级精确提醒，并在系统可能清掉提醒排程后主动补调度，同时把影响提醒触达的关键权限集中展示给用户处理。

## 主要更新

- 每日提醒优先使用 `setExactAndAllowWhileIdle()`，在缺少精确闹钟权限时降级到普通休眠提醒。
- App 启动、回到前台、提醒设置变更时，会自动补一次每日提醒调度。
- 系统重启、应用覆盖升级、时间/时区变化、精确闹钟权限状态变化后，会重新调度每日提醒。
- 设置页每日提醒增加通知权限、准时提醒、通知频道、后台限制状态。
- 用户可从设置页直接跳转到通知权限、闹钟与提醒、通知频道、电池优化相关系统页面。

## 安装与覆盖升级

- 安装包：`DDLAgent-v0.2.6.apk`
- `applicationId` 保持不变：`com.zpc.fucktheddl`
- `versionCode` 已从 `12` 提升到 `13`，可从 v0.2.5 直接覆盖安装。
- 发布包继续通过 GitHub Actions 使用 release keystore 签名。

## 注意事项

- 如果用户在系统设置中“强行停止”应用，Android 会阻止普通应用继续接收提醒，直到用户重新打开应用。
- 国产系统的后台限制仍可能影响提醒触达，建议在设置页按提示放行后台限制。
- 本版本不改变本地数据库 schema，不会迁移或清空已有日程、待办和任务书数据。
