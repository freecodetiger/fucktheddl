# DDL Agent v0.2.5 正式版

v0.2.5 改进了应用内更新体验：当检测到新版本时，用户可以直接在应用内下载安装包、查看下载进度，并在需要时按引导开启系统安装权限。

## 主要更新

- 检查更新现在会读取产品版本元数据中的安装包地址。
- 更新弹窗新增“下载并安装”，下载过程显示实时进度。
- 下载完成后自动打开系统安装器，减少跳转发布页和手动寻找安装包的步骤。
- Android 8 及以上系统如果未允许安装未知来源应用，会引导用户前往系统页面开启“允许来自此来源的应用”。
- 保留发布页面和 GitHub Release 入口，作为下载失败或权限受限时的兜底方式。

## 安装与覆盖升级

- 安装包：`DDLAgent-v0.2.5.apk`
- `applicationId` 保持不变：`com.zpc.fucktheddl`
- `versionCode` 已从 `11` 提升到 `12`，可从 v0.2.4 直接覆盖安装。
- 发布包继续通过 GitHub Actions 使用 release keystore 签名。

## 注意事项

- Android 系统不允许普通应用静默安装 APK，本版本会在下载完成后打开系统安装器，由用户确认安装。
- 本版本不改变本地数据库 schema，不会迁移或清空已有日程、待办和任务书数据。
- 应用内检查更新仍通过 `https://ddlagent.praw.top/version.json` 判断最新版，并使用其中的 `apk_url` 下载安装包。
