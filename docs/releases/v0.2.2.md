# DDL Agent v0.2.2 正式版

v0.2.2 增加了独立的“任务书”分屏，用 RPG 冒险手账的方式承载长期追求。它不参与日程和待办安排，也不接入语音 AI，适合把主线目标、支线目标和头脑风暴拆成可持续维护的树。

## 主要更新

- 新增第四分屏“任务书”，顺序为：日历 / 日程 / 待办 / 任务书。
- 任务书分为主线和支线，每本书都是一棵独立目标树。
- 每本书支持标题、描述、地点、可选预期达成时间。
- 每个小目标也支持标题、描述、地点、可选预期达成时间。
- 树形目标支持无限层级、折叠展示、完成勾选、快速添加子目标。
- 子目标完成不会自动完成父目标，长期目标仍由用户手动判断。
- 待办页右边缘增加两个微弱呼吸箭头，提示右侧还有任务书页面，点击可直接进入。
- 底部导航保持日程 / 录音 / 待办三段，录音按钮继续居中。
- 新增 Room v1 -> v2 数据库迁移，覆盖安装时保留原有日程与待办数据。

## 安装与覆盖升级

- 安装包：`DDLAgent-v0.2.2.apk`
- `applicationId` 保持不变：`com.zpc.fucktheddl`
- `versionCode` 已从 `8` 提升到 `9`，可从 v0.2.1 直接覆盖安装。
- 发布包继续通过 GitHub Actions 使用 release keystore 签名。

## 注意事项

- 任务书是独立模块，不会进入现有统计、每日提醒、日程或待办列表。
- 当前版本不支持拖拽重排节点；可以通过新增、编辑、删除来维护树。
- 任务书不支持语音控制，避免与短句日程/待办输入混淆。
