# DDL Agent v0.2.4 正式版

v0.2.4 修复了两个影响日常编辑体验的问题：日程/待办编辑里的日期与时间选择器无法稳定打开，以及任务书中点击“+ 子目标”后反馈不明显、难以继续向更深层拆解目标。

## 主要更新

- 修复日程日期、开始时间、结束时间选择器点不开的问题。
- 修复待办截止日期选择器点不开的问题。
- 任务书中点击“+ 子目标”后，会在当前节点下方直接展开添加框。
- 如果父节点处于折叠状态，点击“+ 子目标”会先展开父节点，避免新增后看不到子目标。
- 任务书目标树支持继续向更深层添加子目标。
- 调整任务书树形缩进：同一层节点保持同一条垂直线，深层节点不再被压到相同 x 轴位置。

## 安装与覆盖升级

- 安装包：`DDLAgent-v0.2.4.apk`
- `applicationId` 保持不变：`com.zpc.fucktheddl`
- `versionCode` 已从 `10` 提升到 `11`，可从 v0.2.3 直接覆盖安装。
- 发布包继续通过 GitHub Actions 使用 release keystore 签名。

## 注意事项

- 本版本不改变本地数据库 schema，不会迁移或清空已有日程、待办和任务书数据。
- 应用内检查更新仍通过 `https://ddlagent.praw.top/version.json` 判断最新版。
