# DDL Agent v0.2.3 正式版

v0.2.3 补齐了应用内更新入口和 GitHub 仓库说明，让用户可以在设置页直接检查最新版，也能了解项目来源、隐私边界并前往仓库支持项目。

## 主要更新

- 设置页“关于”区域中，点击版本行即可检查更新。
- 检测到新版本后，会提示“点击获取最新版”。
- 更新弹窗提供两个跳转入口：
  - 发布页面：`https://ddlagent.praw.top`
  - GitHub Release：`https://github.com/freecodetiger/fucktheddl/releases`
- 设置页“关于”区域新增 GitHub 仓库入口。
- GitHub 仓库入口使用 GitHub mark 图标。
- 点击仓库入口后先展示制作者签名和隐私说明，再由用户选择是否前往仓库。
- 仓库说明强调：日程、待办和任务书数据保存在本机，应用不依赖独立业务服务器。
- 仓库说明同时提示：DeepSeek 和阿里云配置只用于用户主动发起的模型与语音服务调用。

## 安装与覆盖升级

- 安装包：`DDLAgent-v0.2.3.apk`
- `applicationId` 保持不变：`com.zpc.fucktheddl`
- `versionCode` 已从 `9` 提升到 `10`，可从 v0.2.2 直接覆盖安装。
- 发布包继续通过 GitHub Actions 使用 release keystore 签名。

## 注意事项

- 检查更新需要访问 GitHub Release API。
- 应用不会自动安装更新，只会跳转到发布页面或 GitHub Release 页面，由用户手动下载新版 APK。
